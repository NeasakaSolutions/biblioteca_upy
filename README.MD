# Plataforma Digital de Biblioteca Académica — Universidad Politécnica de Yucatán

## Resumen rápido

Plataforma web académica que permite a docentes compartir material
bibliográfico en formato PDF y a los alumnos acceder, comentar
y organizarse en grupos de estudio.

Desarrollado como parte de prácticas profesionales y servicio social
en la Universidad Politécnica de Yucatán.

- Gestión de usuarios (docentes y alumnos)
- Biblioteca digital con PDFs
- Grupos de trabajo académico
- Stack moderno con Node.js y React
