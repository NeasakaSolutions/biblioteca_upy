# Instalacion de las dependencias:

- Dirigirse a la carpeta del backend en terminal y ejecutar:
```bash
npm install
```

# Dependencias que se utilizaron para este proyecto:

- Express:
```bash
npm install express
```

- Nodemon:
```bash
npm i -D nodemon
```

- Typescript
```bash
npm i -D typescript ts-node
```

- Opcionales:
```bash
npm i --save-dev @types/express
```
